## ğŸ”§ Troubleshooting

### Camera Won't Start

**Problem**: Button says "Loading..." forever

**Solutions**:
1. Refresh the page
2. Check browser permissions (Settings â†’ Privacy â†’ Camera)
3. Try a different browser (Chrome recommended)
4. Ensure no other app is using the camera

---

**Problem**: "Permission denied" error

**Solutions**:
1. Click the camera icon in address bar
2. Allow camera access
3. Refresh the page
4. Try HTTPS connection (required for camera)

### Gestures Not Detected

**Problem**: "Detected Word: None" always

**Solutions**:
1. Improve lighting
2. Move hand closer/farther
3. Ensure full hand is visible
4. Try different hand orientation
5. Check if hand landmarks are visible (purple lines)

---

**Problem**: Wrong gesture detected

**Solutions**:
1. Make gesture more clearly
2. Hold gesture steady for 1-2 seconds
3. Ensure fingers are fully extended/closed
4. Try different hand angle
5. Check gesture guide for correct hand shape

### Performance Issues

**Problem**: Laggy or slow detection

**Solutions**:
1. Close other browser tabs
2. Close other applications
3. Use a more powerful device
4. Reduce browser zoom level
5. Clear browser cache

---

**Problem**: Video feed is choppy

**Solutions**:
1. Check internet connection (for CDN scripts)
2. Reduce other network usage
3. Try lower quality camera setting
4. Close background applications
